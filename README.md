# School Quiz Application

A web-based quiz application for MercyCourt Montessori School with admin dashboard.

## Features
- Student quiz interface
- Admin dashboard for managing questions
- Results tracking
- Image support for questions

## Access
- Student Quiz: https://YOUR-USERNAME.github.io/quiz-app/
- Admin Panel: https://YOUR-USERNAME.github.io/quiz-app/admin.html


## Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla)
- localStorage for data persistence
